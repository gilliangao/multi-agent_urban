# Showcase Artifacts

These files are saved portfolio artifacts from a completed Emergent Canopy run.

- `emergent_canopy_saved_analysis.html`: a static report with recommendations, charts, agent votes, debate transcript, explanations, and the raw audit trace embedded. It does not rerun OpenAI calls.
- `streamlit-page.png`: a visual snapshot of the Streamlit page.

Use these when sharing the project with reviewers or collaborators who should understand the product without needing local data files, API keys, or a running Streamlit server.
